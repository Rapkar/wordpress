this plugin for create ads and receive count  clicks on ads created. can take date and  can reset count clicks. 
== Description ==
user can insert ads and after publish post get shortcode, also if  user using elementor plugin can use of widget ADS.
ADS plugin  can get date of expire ads and show to user,can reset count click, ads not work with user ip,
1-create ads and extra link.
2-create date.
3- add to web(with widget or short code).
4- view the count of clicks. 
5- view expire date of post.
